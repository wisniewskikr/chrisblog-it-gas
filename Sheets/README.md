SHEETS

DESCRIPTION
-----------

##### Goal
The goal of this project is to show how to create and remove sheets using Menu. 

##### Details
When user choose custom menu in Gsheet then Google Apps Script function is run and sheet "Hello World" is created or removed.

##### Result 
Sheet "Hello World" is created or removed.


IMPLEMENTATION
-----------

Implementation details:
* In file "functions.gs" create all functions.
  

LAUNCH
------

To launch project please:
* Open Gsheet;
* Choose "Greeting -> Create Sheet Hello World";
* Choose "Greeting -> Delete Sheet Hello World";
* (Optional) Sometimes you will be asked to confirm permissions.
